🎯 三种使用方式

🎉 刚刚发布成功！ 你的脚手架已经可以从 Maven 中央仓库下载，全世界的人都能用！

方式 1：IDEA 自定义 Catalog（⭐ 推荐 - 干净的列表）

在 IDEA 中 File → New → Project → Maven Archetype：

  1. 点击 "Manage catalogs..."
  2. 点击 "+" → "Add Catalog"
  3. 填写以下两个字段：
Location:
https://jhqxx.github.io/all-package/jhqxx-catalog.xml
Name:
JHQXX 脚手架

✅ 这样 IDEA 列表里 只显示你自己的脚手架，不会混入 Maven Central 的几千个！

方式 2：Maven Central Catalog（默认列表里查找）

使用 IDEA 自带的 Maven Central Catalog：

Catalog URL:
https://repo.maven.apache.org/maven2/archetype-catalog.xml

⚠️ 这个列表里有 成千上万 个脚手架，需要搜索 springai-agent-archetype

方式 3：命令行直接生成

mvn archetype:generate \
  -DarchetypeGroupId=io.github.jhqxx \
  -DarchetypeArtifactId=springai-agent-archetype \
  -DarchetypeVersion=1.0.0 \
  -DgroupId=com.example \
  -DartifactId=my-agent \
  -Dpackage=com.example.agent \
  -DinteractiveMode=false

🚀 快速开始（生成项目后）

# 1. 进入生成的项目目录
cd my-agent

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env，填入 OPENAI_API_KEY

# 3. 运行
./mvnw spring-boot:run

# 4. 测试 API
curl -X POST http://localhost:8080/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"你好"}'

📋 API 端点

  • POST /api/chat - 简单对话
  • POST /api/agent - 智能体对话（带工具调用）
  • POST /api/stream - 流式对话（SSE）
  • POST /api/rag - RAG 知识库问答
  • GET /api/health - 健康检查
