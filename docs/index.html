SpringAI Agent Maven Archetype

📦 Catalog URL

Catalog: https://jhqxx.github.io/ai-siyuan-agent/archetype-catalog.xml

🚀 在 IntelliJ IDEA 中使用

  1. 打开 File → New → Project
  2. 选择 Maven 模板
  3. 勾选 "Create from archetype"
  4. 点击 "Add Archetype..."
  5. 填写：
    • GroupId: io.github.jhqxx
    • ArtifactId: springai-agent-archetype
    • Version: 1.0.0
    • Repository: https://jhqxx.github.io/ai-siyuan-agent/maven-archetype/repo/

🛠 使用 Maven 命令行

mvn archetype:generate \
  -DarchetypeGroupId=io.github.jhqxx \
  -DarchetypeArtifactId=springai-agent-archetype \
  -DarchetypeVersion=1.0.0 \
  -DgroupId=com.mycompany \
  -DartifactId=my-agent \
  -DarchetypeRepository=https://jhqxx.github.io/ai-siyuan-agent/maven-archetype/repo/

✨ 包含内容

  • Spring Boot 3.3 + Spring AI 1.0
  • ChatClient Bean 配置
  • 3 个示例工具：WeatherTool, CalculatorTool, DateTimeTool
  • RAG 服务（自动加载 data/*.txt）
  • REST API: /api/chat, /api/agent, /api/rag, /api/stream
  • 流式对话（SSE）支持
