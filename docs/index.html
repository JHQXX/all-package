方式 1：使用 GitHub 模板（推荐 ⭐）

在 IDEA 的 New Project 对话框中：

  1. 选择 Maven Archetype 模板
  2. 点击 "Manage catalogs..." 添加 Catalog

🚀 快速开始（基于模板项目）

# 1. 通过 "Use this template" 创建仓库后，克隆到本地
git clone https://github.com/你的用户名/你的项目名.git
cd 你的项目名

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env，填入 OPENAI_API_KEY

# 3. 运行
./mvnw spring-boot:run

# 4. 测试
curl -X POST http://localhost:8080/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"你好"}'

📋 API 端点

  • POST /api/chat - 简单对话
  • POST /api/agent - 智能体对话（带工具调用）
  • POST /api/stream - 流式对话（SSE）
  • POST /api/rag - RAG 知识库问答
  • GET /api/health - 健康检查
